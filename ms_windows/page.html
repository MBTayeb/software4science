Since more than 70% of all computers worldwide run on Microsoft Windows, we will discuss here some relevant things for those using it as their operating system. Unfortunately, many programming tools are designed for Unix-based systems, which can cause a lot of trouble and waste a lot of time. We will list here some tools that can make your life much easier as a Windows user. We recommend using only supported versions of Windows, because many tools are not available for older versions. You can check whether your version is supported in the Windows release health page https://learn.microsoft.com/en-us/windows/release-health/ .
