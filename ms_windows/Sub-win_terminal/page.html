Windows Terminal and prompt customization

Windows Terminal is a must-have as a Windows user. It is a terminal emulator and an improvement of the old Windows Console. It brings the CLI experience to another level.

In addition to that, Oh-My-Posh can make your experience even more enjoyable. It customizes the prompt you see in the terminal very easily. You can pick one of many themes. Since it changes the prompt, each shell (PowerShell, cmd.exe, Bash ...) used within the computer needs a separate setting. Typically, people add it to the shell profile, which is a file used to configure what should be loaded in the start of a new shell session. If one wants to use it within a WSL, one would need to install it in WSL.
