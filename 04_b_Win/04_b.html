Since more than 70% of all computers worldwide run on Microsoft Windows, we will discuss here some relevant things for those using it as their operating system. Unfortunately, many programming tools are designed for Unix-based systems, which can cause a lot of trouble and waste a lot of time. We will list here some tools that make your life much easier as a Windows user.

Windows Terminal

Windows Terminal is a must-have as a Windows user. It is a terminal emulator and an improvement of the old Windows Console. It brings the CLI experience to another level.

Powershell

Windows Powershell is a built-in command line shell in the Windows OS. What differentiates it from the other Windows command line shell "cmd.exe", also known as "Command Prompt", is the fact that it is at the same time a scripting language (like Python e.g.) and also a configuration management framework (it can be used to automatically track and control many computers at once). Thus, it can do everything a cmd can do and much more. This is why it has been named Powershell.

Powershell can be compared to the Linux Bash shell. However, it is systematically different. The output of Bash commands is in text format, while the output of Powershell commands is in object format. This can cause a lot of confusion for those who are used to Bash. Moreover, all the commands in Powershell are named in the form of "verb-noun", which could be helpful for the beginners. This is very different from bash, where the commands are usually one word. For example, "Get-ChildItem" is the equivalent of "ls" in bash. Still, there are many aliases in Powershell that rename the commands to their Bash equivalent ("ls" is available in Powershell too).
