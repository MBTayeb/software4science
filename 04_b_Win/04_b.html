Since more than 70% of all computers worldwide run on Microsoft Windows, we will discuss here some relevant things for those using it as their operating system. Unfortunately, many programming tools are designed for Unix-based systems, which can cause a lot of trouble and waste a lot of time. We will list here some tools that can make your life much easier as a Windows user.

Package Managers

Whether one is used to the Linux Packet managers or not, package managers can save much time. There are three main one available on Windows, i.e. Winget, Chocolatey and Scoop. Winget is made by Microsoft and very simple to use. Chocolatey is made by the community and has more packages available. However, it must first be installed and needs an elevated shell for each package installation. We recommend using Winget, since it should be enough for most users.

Powershell

Windows Powershell is a built-in command line shell in the Windows OS. What differentiates it from the other Windows command line shell "cmd.exe", also known as "Command Prompt", is the fact that it is at the same time a scripting language (like Python e.g.) and also a configuration management framework (it can be used to automatically track and control many computers at once). Thus, it can do everything a cmd can do and much more. This is why it has been named Powershell.

Powershell can be compared to the Linux Bash shell. However, it is systematically different. The output of Bash commands is in text format, while the output of Powershell commands is in object format. This can cause a lot of confusion for those who are used to Bash. Moreover, all the commands in Powershell are named in the form of "verb-noun", which could be helpful for beginners. This is very different from bash though, where the commands are usually one word. For example, "Get-ChildItem" is the equivalent of "ls" in bash (short for list). Still, there are many built-in aliases in Powershell that rename the commands to their Bash equivalent (e.g. "ls" works also on Powershell).

Starting from the 6th version, Powershell has become drastically different from the previous versions. So the newer ones are called Powershell Core to differentiate them from the old ones, i.e Windows Powershell. The Core versions are open-source and available on macOS and Linux too.

WSL

Windows Subsystem for Linux is a Microsoft service that allows you to run a Linux OS within Windows. This has been a revolutionary step from Windows, as it allows one to have the best of both Worlds. This has improved further with the WSL 2, with which one can open many GUIs. There are many WSL distributions available directly in the Microsoft Store.
