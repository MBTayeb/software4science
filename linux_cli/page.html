A lot of work in RSEng (and arguably most of it in science-related RSEng) is done under Linux and making use of command line interfaces (CLIs). You should therefore try to get familiar with both before starting to work on the following sections.

You can find our own tutorial linked above in this chapter. Topics include:
  • What is Linux?
  • How to run Linux?
  • What is a CLI?
  • What are Vi and SSH used for?
  • Where to find help?
So, here are the online resources which will help you to learn all of this:
