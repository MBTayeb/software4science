This part provides background information on RSEng which serves as a first overview of the topic.

RSEng is concerned with all technical, intellectual and also social issues concerning the development of software in scientific contexts. Important topics are programming languages, programming tools, etc. but also the challenges we face when 'translating' theoretical considerations with a scientific background (a 'scientific theory') into algorithms that can be implemented as source code for software, as well as issues like the reproducibility, sharing of code etc.
