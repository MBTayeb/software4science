Templates

When starting a programming project, there are a certain number of things that need to be done. One should for example make the project structure, write licensing files, make Coding files... It can save a lot of time to just use a template, instead of starting anew. Cookiecutter is a tool that lets you make "cookiecutters", i.e. a project templates.

There are also Github template repositories, which offer a similar service to cookiecutters. What makes Github different is the fact that your new project is from the very beginning hosted on the platform.

https://github.com/cookiecutter/cookiecutter
-- Tool "that creates projects from cookiecutters (project templates)"
https://github.com/cookiejar/cookietemple
-- Tool that uses cookiecutter to make cookiecutters from a chosen repository
https://www.ssc.uni-heidelberg.de/en/learning/coding-templates
-- GitHub and Cookiecutter templates for Python, C++ and Fortran provided by the University of Heidelberg
https://github.com/UtrechtUniversity/rse-project-templates
-- GitHub and Cookiecutter templates for RSE provided by Utrecht University

Copilot and LLM

Large Language Models (LLMs) have changed the world of developing. You can now with the help of an LLM write code with a programming language that you have never learned before. Of course, LLMs have their limit, but they are really useful when one is developing or learning a new programming language.

Copilots are editor extensions (usually with subscription only) that use AI to give you coding suggestions. Their strength lies within the fact that they learn from your code files and from your usage.
