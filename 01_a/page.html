Software for Science

A self-study course offered by the IVV NWZ (Informationsverarbeitungs-Versorgungseinheit Naturwissenschaften)
(with corrections and contributions from Carsten Kemena and Daniel Nüst - thanks a lot!)
The current second version of the course was to a large extend prepared by:: Tayb Marrakchi-Benjaafar

This self-study course guides the student in a structured manner through a number of external video, audio, text, and software resources, so that he will be able to learn software engineering skills for the use in Research on his own. Example topics are introduction to Linux, using command line interfaces,  editors/IDEs, compilers, software libraries, build and version control systems, source code repositories, as well as the debugging, testing and optimization of scientific software.


A Zoom-Workshop is offered as a starting point for this self-study course.

In addition, the IVV NWZ offers a one-day crash course in RSEng based on (part of) the material shown here for graduate schools, institutes, etc.

If you would like to hear this course or organize a crash course, please get in touch with Martin Korth via dgd@uni-muenster.de.

People involved in teaching at the University of Münster should feel free to use the material provided here for their teaching activities.
Please also get in touch with Martin if you want to contribute to this self-study course!
And why not join the AG RSE at University of Münster? Just get in touch via rse@uni-muenster.de!
Note:

As a learner going through the chapters of this course, you are not supposed to study all the materials. Many of the links that are listed here are for those who would like to get deeper knowledge in a specific theme. The links that we most recommend will be in bold.


Overview of the course:
  • Introduction and Overview
  • Motivation
  • Organisations, General Online-Resources, ...
  • Computer Know-How
  • Introduction to Linux, Using a Command Line Interface (CLI)
  • Microsoft Windows, Powershell, Windows Subsystem for Linux (WSL)
  • Programming Languages
  • Programming Paradigms
  • Editors, Integrated Development Environments (IDEs)
  • Compilers, Software Libraries, Build Systems
  • Version Control Systems (git), Source Code Repositories
  • Continuous Integration
  • Containers
  • Debugging, Testing
  • Templates, Copilot
  • Optimization, Style
  • Advanced RSEng Topics
