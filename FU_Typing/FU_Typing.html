10 finger typing, or touch typing, is an awesome skill that can save time for anyone using a computer. The goal is basically being able to type by only touching the keyboard, that is, without looking at it. Optimally, one should use as many fingers as possible. But using a bit less than 10 fingers is often to type with a decent speed. An essential thing to keep in mind is to always type each letter with the same finger.
Work through one of these pages for maybe 10 minutes each day for one semester (do it right after reading Spiegel Online in the morning!) and you will be done. Make sure to create an account in the website you choose so that you set the language and the keyboard layout you want to practice on. This also allows you to see statistics about your typing habits.
