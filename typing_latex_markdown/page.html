Typing

10 finger typing, or touch typing, is an awesome skill that can save time for anyone using a computer. The goal is basically being able to type by only touching the keyboard, that is, without looking at it. Optimally, one should use as many fingers as possible. But using a bit less than 10 fingers is often to type with a decent speed. An essential thing to keep in mind is to always type each letter with the same finger.
Work through one of these pages for maybe 10 minutes each day for one semester (do it right after reading Spiegel Online in the morning!) and you will be done. Make sure to create an account in the website you choose so that you set the language and the keyboard layout you want to practice on. This also allows you to see statistics about your typing habits.

LaTeX

LaTeX is a markup language made for making documents like Microsoft Office Word. It is the standard way for writing papers in many fields, especially STEM fields. Writing mathematical equations in LaTeX is quite simple, which is probably the reason for its popularity in these fields. In LaTeX, what you see is, unlike MS Word, not what you get. The desired result can't be seen before compilation. Using LaTeX comes with many advantages compared to MS Word, things like bibliography, list of figures or tables and referencing are done with ease. LaTeX also allows to customize pretty much anything in the document.
Since LaTeX has a lot of similarities with programming languages, it requires some time to learn how to make beautiful LaTeX documents. However, learning some basics should be enough to get started. One should then look for help, whenever there is need to. Websites like Stack Exchange provide helpful answers for most questions any beginner has. (Of course, advanced questions also have answers there.) It is quite recommended to begin with writing LaTeX on Overleaf, as it makes the workflow much easier. A more advanced LaTeX user might want to install a LaTeX distribution for more flexibility.

Markdown

Just like LaTeX, Markdown is a markup language. In other words, it is a system made for formatting and structuring a document. What makes it different from LaTeX or other markup languages (like HTML etc...) is it simplicity. Eventhough Markdown code is easier to read and write, it looks proffesional once rendered. This is what makes it so popular. A famous use of Markdown is in Github README files. People use it for making websites, writing a book and many other things.
