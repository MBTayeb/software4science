Jupyter Notebooks (formerly IPython) is a web-based environment for writing and executing Jupyter notebook JSON (.ipynb) files. It offers input/output cells with code, text incl. mathematical formulas, plots, media, which in addition can be converted to standard output formats like HTML, LaTeX, PDF, Markdown, Python etc. It is arguably the most commonly used environment for solving data science tasks in Python. Alternative supported languages include R, Julia etc. The kernel of  a Jupyter notebook is the computational (usually cloud) resource to execute code. Each cell of a notebook is a container for code to be executed, or text/graphs/urls to be shown, etc. Using Markdown syntax, text can be formatted.

The Jupyter Notebook at the bottom of this work unit illustrates how Jupyter Notebooks work.

At WWU there is a 'Jupyter Hub', a multi-user server for Jupyter notebooks sessions as ’pods’ in the Kubernetes cluster 'ZIVkube' on the WWU Cloud. When logging in you can choose CPU/RAM resources and a notebook 'image' with pre-installed features for either eScience, software development, etc. From these notebooks you have access to the WWU HPC system Palma II, as well as the WWU Cloud. Your JupyterHub home is accessible as network share within the WWU network.
