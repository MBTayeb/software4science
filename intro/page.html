Software for Science

A self-study course

This self-study course guides the student in a structured manner through a number of external video, audio, text, and software resources, so that he will be able to learn software engineering skills for the use in Research on his own. Example topics are introduction to Linux, using command line interfaces, editors/IDEs, compilers, software libraries, build and version control systems, source code repositories, as well as the debugging, testing and optimization of scientific software.

The current version of the course was to a large extend prepared by Tayb Marrakchi-Benjaafar under the supervision of Martin Korth. The course was fonded by the IVV NWZ, which is the Information Processing and Supply Unit for the faculty of Biology, the faculty of Chemistry and Pharmacy and the faculty of Physics within the University of Münster.

Important corrections and contributions were made by Carsten Kemena and Daniel Nüst. Thanks a lot!
