IVV NWZ self-study course Research Software Engineering (RSEng)

(with corrections and contributions from Carsten Kemena and Daniel Nüst - thanks a lot!)

This self-study course guides the student in a structured manner through a number of external video, audio, text, and software resources, so that he will be able to study the topic of Research Software Engineering on his own. Example topics are introduction to Linux, using command line interfaces,  editors/IDEs, compilers, software libraries, build and version control systems, source code repositories, as well as the debugging, testing and optimization of scientific software.

People involved in teaching at WWU should feel free to use the material provided here for their teaching activities. In addition, IVV NWZ offers a one-day crash course in RSEng based on (part of) the material shown here for graduate schools, institutes, etc. – for this, please get in touch with Martin Korth ( dgd@uni-muenster.de ). Please also get in touch with Martin if you want to contribute to this self-study course!

Overview of the course:
  1. Introduction and Overview
  2. Background Information (organisations, general online-resources, …)
  3. Basic Computer Know-How
  4. Introduction to Linux, Using Command Line Interfaces (CLI)
    • Excursion: Bash vs Powershell ... upcoming ...
  5. Programming Languages, Editors, Integrated Development Environments (IDEs)
  6. Compilers, Software Libraries, Build Systems
  7. Version Control Systems, Source Code Repositories
  8. Debugging, Testing, Optimization
  9. Advanced RSEng Topics
  10. Additional Resources
  11. Focus unit: Jupyter Notebooks
  12. Focus unit: Python / OOP
  13. Focus unit: Julia
  14. Focus unit: R
  15. Focus unit: Software & Data Citation & Licensing (author: Daniel Nüst)

There are separate IVV NWZ self-study courses on Research Software Engineering, Research Data Management, Data Science/Machine Learning, Bioinformatics/Statistics with R and upcoming Linux Administration as well as Research Data Infrastructure/Cloud Computing, i.e. Docker/Kubernetes/WWU-Kube, for more information see here: https://www.uni-muenster.de/NWZ/en/Angebot/Lehre/index.html

Note: As a learner going through the chapters of this course, you are not supposed to visit all the links. Many of the links that are listed here are for those who would like to get deeper knowledge in a specific theme.
